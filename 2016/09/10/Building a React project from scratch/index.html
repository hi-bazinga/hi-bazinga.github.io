Configuring webpack-dev-server

So far, we’ve created a very simple React application by using npm, webpack and Babel. But it may be very troublesome that we have to rebuild the bundle every time when new changes come, especially during the rapid development process.

A simple solution is “watch mode” in webpack. In this mode, webpack will watch all dependencies and recomplie on change. Just add --watch when executing webpack:

Now every change you do will trigger a recompilation, and we can see the changes after refreshing the page.
